# ASR_with_Nemo
En este repositorio encontraras los pasos y archivos necesarios para entrenar un modelo de Reconocimiento de voz.

Las librelias necesarias serán:

- Nemo (ver instalación)
- Pytorch
- Pytorch lightning
- libsndfile1
- ffmpeg
- Librosa
- Pandas
